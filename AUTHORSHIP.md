# Authorship and development process

This project was conceived, directed, reviewed, and published by Fernando Bitti Loureiro.

AI-assisted development tools, including Replit and Devin, were used as software assistants during portions of the development process. Their use included code suggestions, iteration support, and implementation assistance.

Human authorship remained central to the final work, including:

- project concept and product direction
- selection of features and educational goals
- architectural choices
- acceptance, rejection, and revision of generated code
- visual, textual, and UX decisions
- final review and publication

This file is intended to document the development process and authorship record. It does not reduce or waive any copyright, trademark, or other rights in the project.
